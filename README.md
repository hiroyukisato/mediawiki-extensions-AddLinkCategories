#AddLinkCategories
## 概要
ページ内リンクのclassにリンク先のカテゴリを追加する。

## インストール
LocalSettings.php

    wfLoadExtension( 'AddLinkCategories' );

## 設定[

classに書き込むカテゴリを MediaWiki:AddLinkCategories-Whitelist で指定する

```
* [[:カテゴリ:キュートのメンバー]]

```
